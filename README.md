# Prof Sénégal

## Description

Prof Sénégal est une application web conçue pour aider les professeurs de lycée au Sénégal à centraliser et organiser leur travail. L'application permet aux enseignants de gérer leurs emplois du temps, classes, devoirs et documents de manière simple et efficace.

## Fonctionnalités

- **Gestion des établissements** : Ajoutez plusieurs établissements (publics et privés) et gérez les classes enseignées.
- **Emploi du temps intelligent** : Créez et visualisez votre emploi du temps hebdomadaire avec des couleurs par établissement ou matière.
- **Gestion des classes et matières** : Ajoutez des classes et associez-les à des matières spécifiques.
- **Devoirs et corrections** : Créez des devoirs, suivez leur statut de correction et gérez les fichiers associés.
- **Stockage cloud des documents** : Téléchargez et organisez vos documents (cours, exercices, corrigés) dans le cloud.
- **Recherche rapide** : Retrouver facilement vos fichiers grâce à une fonction de recherche intégrée.
- **Accessibilité** : Application accessible en ligne sur PC et mobile, avec une fonctionnalité hors ligne partielle.

## Technologies utilisées

- **Frontend** : React.js avec Material UI ou Ant Design pour une interface moderne et réactive.
- **Backend** : Node.js avec Express pour la gestion des API et des données.
- **Base de données** : MySQL pour le stockage des données.
- **Application Web Progressive (PWA)** : Fonctionne comme une application mobile, avec possibilité d'installation sur Android sans Play Store.

## Installation

1. Clonez le dépôt :
   ```
   git clone <url-du-dépôt>
   ```
2. Accédez au répertoire du client :
   ```
   cd client
   ```
3. Installez les dépendances :
   ```
   npm install
   ```
4. Accédez au répertoire du serveur :
   ```
   cd ../server
   ```
5. Installez les dépendances :
   ```
   npm install
   ```

## Lancement de l'application

- Pour démarrer le client :
  ```
  cd client
  npm start
  ```

- Pour démarrer le serveur :
  ```
  cd server
  npm start
  ```
